Description

This HTML program creates a Daily Meal Plan using ordered and unordered lists.
It displays different meal sections such as:

Breakfast
Lunch
Dinner

Each meal contains:

Main items
Side items
Additional sub-items using nested lists

The program demonstrates:

HTML headings (<h1>, <h2>)
Ordered lists (<ol>)
Unordered lists (<ul>)
Nested lists structure
Features
Organized meal categories
Nested list representation
Simple and beginner-friendly HTML structure
Easy to customize with new food items
Technologies Used
HTML5
Purpose

This project is useful for beginners to learn:

List creation in HTML
Nested lists
Basic webpage structure formatting
Output

The webpage displays a structured daily meal schedule with breakfast, lunch, and dinner sections.
